Zbalermorna Font Examiner

This page has contains various featues of the Zbalermorna typeface, and can be used to preview fonts during development. If you've merged the src/zlm.fea feature file into your font, you should be able to copy text directly from this page into the metrics window in FontForge to modify the kerning/spacing if you see that something is off. Check out the Javascript Zbalermorna IME to test out everything in real-time. You can also use an OS IME to input text as well.
